API DOCUMENTATION

VECTOR CRASHER API

All endpoints require `Authorization: Bearer <token>` except login/register/admin login
Authentication
POST /api/register
Register a new user (requires admin approval).
Body: { "email": "user@gmail.com", "phone": "447123456789", "age": 25, "password": "secret" }
POST /api/login
Login and receive JWT token.
Body: { "email": "user@gmail.com", "password": "secret", "remember": true }
Response: { "success": true, "token": "jwt_token" }
POST /api/forgot-password
Request password reset link.
Body: { "email": "user@gmail.com" }
POST /api/reset-password
Reset password with token.
Body: { "token": "reset_token", "newPassword": "newpass" }
User Profile
GET /api/user/profile
Get current user profile.
GET /api/user/status
Get user tier, expiration, premium status.
POST /api/user/update-profile
Update age, gender, country, phone.
Body: { "age": 26, "gender": "Male", "country": "USA", "phone": "+1234567890" }
POST /api/user/change-password
Change user password.
Body: { "oldPassword": "old", "newPassword": "new" }
POST /api/user/upload-avatar
Upload profile picture (multipart/form-data, field: "avatar").
POST /api/user/delete-account
Permanently delete account.
GET /api/user/device-status
Check if WhatsApp device is connected.
WhatsApp Pairing
GET /code?number=447123456789
Request a pairing code for the given WhatsApp number (returns 8-digit code).
GET /qr
Generate a QR code for linking a new device (returns dataURL).
Bug Arsenal
GET /api/bugs
List all available bugs (premium-only flag included).
POST /api/bugs/execute
Execute a bug on a target.
Body: { "bugId": "bug-uuid", "target": "+447123456789" }
Admin
POST /api/admin/login
Admin login (password: U can not know).
Body: { "password": "ZentrixTechOfficial" }
GET /api/admin/stats
Get system statistics (users, approved, premium).
GET /api/admin/users
List all users.
POST /api/admin/approve-user
Approve a pending user.
Body: { "userId": "user-id" }
POST /api/admin/update-tier
Update user tier (lite/premium) and extend expiry.
Body: { "userId": "...", "tier": "premium", "daysToAdd": 30 }
POST /api/admin/delete-user
Delete user account.
Body: { "userId": "..." }
POST /api/admin/create-user
Create a user manually (approved immediately).
Body: { "email": "...", "phone": "...", "age": 25, "password": "...", "tier": "premium", "daysValid": 30 }
POST /api/admin/broadcast
Send a broadcast message to users.
Body: { "message": "text", "target": "all|approved|premium" }
POST /api/admin/create-channel
Create a channel (public feed).
Body: { "name": "channel", "description": "desc" }
POST /api/admin/delete-channel
Delete a channel.
Body: { "channelId": "..." }
POST /api/admin/create-group
Create a group (premium badge if premium=true).
Body: { "name": "group", "premium": true }
POST /api/admin/delete-group
Delete a group.
Body: { "groupId": "..." }
POST /api/admin/add-bug
Add a new bug (executable code).
Body: { "name": "bug", "category": "group", "description": "...", "targetPlaceholder": "...", "icon": "fas fa-bug", "premiumOnly": true, "code": "..." }
POST /api/admin/delete-bug
Delete a bug.
Body: { "bugId": "..." }
Social (Chat, Groups, Channels)
GET /api/channels
List all channels (read‑only).
GET /api/groups
List groups the user is a member of.
POST /api/create-group
Create a normal group (any user).
Body: { "name": "group name" }
POST /api/join-group
Join a group.
Body: { "groupId": "..." }
POST /api/leave-group
Leave a group.
Body: { "groupId": "..." }
GET /api/messages?chatType=dm|group|channel&chatId=...
Retrieve messages for a chat (DM, group, channel).
POST /api/send-message
Send a text or media message.
Body: { "chatType": "dm", "chatId": "recipient@email.com", "text": "hello", "mediaUrl": "optional" }
POST /api/upload-chat-media
Upload image/video/audio for chat (multipart, field: "media"). Returns URL.
